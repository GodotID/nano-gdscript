# nano-gdscript
GDScript Syntax Highlighting in GNU Nano. Updated regularly every minor updates.
Contributions are welcomed

# Installation
This is 100% free, proof:
1. Admit that `install` is an executable
```sh
chmod +x install
```
2. Start the installation
```sh
./install
```
3. Test it
```sh
nano test.gd
```
